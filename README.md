# flask-api
rest api using sqlalchemy

### Steps to Install

     git clone 
     cd venv/scripts
     activate
     #workspace venv    
     pip install -r requirements.txt

     python migrate.py db init
     python migrate.py db migrate
     python migrate.py db upgrade
     python run.py
